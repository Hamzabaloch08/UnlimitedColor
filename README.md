# Background Color Changer

A simple web application that dynamically changes the background color of a display box to a random hex color every second when the "Start" button is clicked. The "Stop" button stops the color change.

## Features

- **Dynamic Background Color**: The background color of the display box changes every second.
- **Hex Code Display**: The current hex color code is displayed inside the color box.
- **Start/Stop Buttons**: Controls to start and stop the background color change.
